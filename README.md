# GoKD

Go library for Windows kernel and user-mode debugging, built on top of the DbgEng engine
(`dbgeng.dll`) — the same engine that powers `kd.exe`, `cdb.exe`, and `windbg.exe`.

See [PLAN.md](PLAN.md) for the full architecture and implementation plan.

## Status

Early development. See PLAN.md for phased delivery roadmap.
